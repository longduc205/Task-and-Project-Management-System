## The Project Manager

![Image](/Users/azure_awabi/Downloads/Task-and-Project-Management-System/markdown_docs/12 Project roles, stakeholders and leadership, teamwork_artifacts/image_000002_de8fdcd317be0b34fd51167afa59a184867c3a48d5bfb3776c4ca51677d19c7c.png)

- Leads the entire project team.
- Integrates cross-functional work efforts.
- Manages schedule, budget, and scope.
- Requires strong technical, soft skills.
- Resolves issues and removes roadblocks.
- Acts as central communication hub.

## Conflict Resolution Process

![Image](/Users/azure_awabi/Downloads/Task-and-Project-Management-System/markdown_docs/12 Project roles, stakeholders and leadership, teamwork_artifacts/image_000013_cffacfddb3ce296c88026469c54a3beadb471cb8f5026ec16a2e520ed0536484.png)

1. Identify the root causes.
2. Gather facts from everyone.
3. Facilitate open group discussion.
4. Focus on issues, not personalities.
5. Seek collaborative win-win resolutions.
6. Document agreements immediately.
