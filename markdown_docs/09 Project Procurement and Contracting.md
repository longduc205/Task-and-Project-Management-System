## The Project Contract

- Formal agreement between parties.
- Defines scope and schedule.
- Establishes terms of payment.
- Allocates project financial risk.

## Fundamental Contract Types

## Fixed Price (FP)

- Lump sum for project.
- Price remains strictly fixed.
- Contractor assumes financial risks.

## Cost-Plus (CPFF)

- Reimbursement for all costs.
- Plus fixed contractor fee.
- Customer assumes financial risks.
