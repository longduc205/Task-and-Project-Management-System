## Accuracy vs.Precision

- Accuracy means closeness to actuals.
- Precision means exact decimal points.
- Goal: Maximize actual estimate accuracy.
- Use PERT three-point estimates.
- Combines optimistic, pessimistic, and likely.

## Project Cost Accounting Systems

- PCAS tracks all project costs.
- Aggregates work package cost estimates.
- Tracks actual versus budgeted costs.
- Quickly identifies any cost variances.
- Essential for largeIT projects.

<!-- image -->

## Life Cycle Costs (LCC)

- Looks beyond project execution phase.
- Cradle-to-grave system costs.
- Includes operations and maintenance costs.
- Maintenance costs often exceed project.
- Decisions today impact future costs.
