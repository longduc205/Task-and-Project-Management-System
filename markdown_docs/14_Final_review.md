## ProjectRoles,Stakeholders,Leadership,&amp;Teamwork

## Project Roles &amp; Stakeholders

- Project Manager (PM): Responsible for planning, execution, and closing.
- Product Owner (PO): Defines requirements and prioritizes backlog.
- Team Members: Execute tasks and deliverwork.
- Stakeholders: Internal/external parties interested in project outcome.

## Leadership&amp;Teamwork

- Leadership Styles: Directive, supportive, servant leadership.
- Teamwork Essentials: Communication, collaboration, trust.
- Shared Goals: Aligning individual efforts with project objectives.
- Conflict Resolution: Addressing issues constructively.
